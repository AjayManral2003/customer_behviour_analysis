# 📊 Customer Behavior & Sales Analytics Project

## 📝 Overview
This project demonstrates an end-to-end Data Analytics workflow. I have performed data cleaning and EDA in **Python**, executed advanced queries in **SQL**, and built an interactive dashboard in **Power BI** to find actionable business insights.

## 🛠 Tools & Technologies
* **Python:** Pandas, NumPy (Data Cleaning & EDA)
* **SQL:** MySQL / PostgreSQL (Data Querying)
* **Power BI:** Interactive Dashboarding
* **Gamma:** Professional Presentation & Reporting

## 🚀 Project Workflow
1. **Data Loading & Cleaning (Python):** Handled missing values, duplicates, and formatted data for analysis.
2. **SQL Analysis:** Imported cleaned data to run queries on customer segments, revenue trends, and KPI calculations.
3. **Power BI Dashboard:** Visualized key metrics like total sales, customer demographics, and product performance.
4. **Final Presentation:** Summarized all insights into a professional report using Gamma.

## 📈 Key Results
* Identified top-performing product categories.
* Found patterns in customer shopping behavior based on discounts.
* Visualized regional sales distribution for better marketing focus.

## 📂 How to Run
* Open `.ipynb` for the Python cleaning process.
* Import the `.csv` into your SQL environment and run the `.sql` scripts.
* Open the `.pbix` file in Power BI Desktop to view the dashboard.
